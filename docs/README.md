# Yakt Documentation

This directory contains the documentation site for Yakt, built with [VitePress](https://vitepress.dev/).

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Structure

- `index.md` - Homepage
- `guide/` - User guides
- `api/` - API reference
- `examples/` - Code examples
- `.vitepress/config.js` - VitePress configuration

## Deployment

The docs can be deployed to GitHub Pages, Netlify, Vercel, or any static hosting service.

For GitHub Pages, configure the base path in `.vitepress/config.js` to match your repository name.
